# Tutor-Matching-Web-App

[Zobacz film demo aplikacji](DEMO.mp4)

## Wymagania techniczne ##
 * Java 17+
 * Node.js 18+
 * Angular

## Instrulcja uruchomienia ##

1. ``` bash
   cd TutorMatch
   ```
2. Zbudowanie projektu
   ``` bash
   mvnw.cmd clean install
   ```
3. Uruchomienie backendu
   ``` bash
   mvnw.cmd spring-boot:run
   ```
4. ``` bash
   cd tutormatch-frontend
   ```
5. Zainstalowanie zależności
   ``` bash
   npm install
   ```
6. Uruchomienie aplikacji
   ``` bash
   ng serve
   ```
